    Resources#

    Resources are values that players accumulate throughout the game.

    Syntax#

    resource identifier {
  name: "Display Name"
  icon: "X"
  initial: 0
}

    Properties#

    Property Type Required Default Description
    name string Yes - Display name shown in UI
    icon string Yes - Icon/emoji shown with resource
    initial number No 0 Starting amount

    Examples#

    Basic Resource#

    resource gold {
  name: "Gold"
  icon: "$"
  initial: 0
}

    Multiple Resources#

    resource coins {
  name: "Coins"
  icon: "$"
  initial: 100  # Start with 100 coins
}

resource gems {
  name: "Gems"
  icon: "*"
  initial: 0
}

resource wood {
  name: "Wood"
  icon: "#"
  initial: 0
}

    Using Emoji Icons#

    resource cookies {
  name: "Cookies"
  icon: "üç™"
  initial: 0
}

  name: "Gold"

    Usage#

    Resources are referenced by their identifier in other blocks:

    resource gold {
  name: "Gold"
  icon: "$"
  initial: 0
}

# Reference the resource by its identifier
action mine {
  produces: 1 gold
}

generator miner {
  cost: 10 gold
  produces: 1 gold
}

    Display#

    Resources appear in the resource bar at the top of the game screen, showing:

    • Icon
    • Name
    • Current amount
    • Production rate (per second)
